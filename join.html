  • 一. 产品运营经理

    • 职位描述：

    1. 分析受众需求，参与设计产品功能，根据产品特性进行运营规划，制定产品优化方案，整体协调平台的运营工作，重点提升用户活跃度和留存，提升营运效益，确保运营目标的实现；

    2. 针对运营目标，根据市场需求，搭建相应的数据统计系统，深入分析用户行为，构建完善的数据监控机制，真实反应产品运营状况，为产品发展提供决策依据；

    3. 梳理线上产品相关的业务流程，优化业务流程在线上产品中的表现形式，从用户的生命周期出发，推进产品的持续优化，对产品体验提出优化建议和需求；

    4. 挖掘公司内容产品与互联网的结合点，参与设计具有建设性的运营方案，管理运营项目的执行过程，与其它部门同事共同配合达成目标；

    5. 具有线上线下活动策划的经验和能力，需要有成功案例。

    职位要求：

    1. 统招本科及以上学历，计算机、数学、统计类专业优先，985/211院校优先；

    2. 3年以上互联网产品运营经验，教育行业的经验优先；

    3. 有过0到1的产品运营经验，对数据敏感，具有较强的数据分析能力；

    4. 拥有很强的执行力，主观能动性强，有责任感，能够承受较强的工作压力；

    5. 具有系统的移动产品运营管理策略和方法论；

    6. 思维活跃，对移动互联网产品的市场动向的敏感度高，善于沟通和协调资源；

    7. 有良好的沟通能力和团队协作能力。

  • 二. 电商运营经理

    • 职位描述：

    1. 负责公司线上商城的运营管理，制定商城的整体运营方案、目标及执行策略，并能够将运营方案进行系统的分解和执行；

    2. 根据商城的规划制定具体的销售目标，并协调公司内外部的资源，进行日常运营工作的计划、活动计划、推广计划的制定，并能够有效的跟踪每个环节的执行情况；

    3. 对线上店铺日常数据直接负责，并形成报表，明确运营过程中出现的问题和原因，并提出可行的改进策略；

    4. 搭建课程体系、规划年度产品监管课程的制作、保证课程质量；

    5. 培训、考核和评估课程开发师的工作；

    6. 同线下产品部协调配合，制作相应的课程内容；

    7. 协同产品销售深度分析用户需求和行为习惯，并根据分析的结果制定商城、产品、详情以及品牌形象的展现策略并实施；

    8. 对商城全部环节的用户体验、业务流程及团队绩效进行分析和不断改进，明确业务流程中的各个关键点，并提出流程优化和关键点优化的方案并实施；

    9. 熟悉商城各个环节中应该注意的事情，从流量来源、转化、复购，负责策划商城的日常活动、节假日活动；

    10.负责店铺运营的各项技术方案及实施方案的制定、执行。

    任职要求：

    1. 统招本科以上的学历，985/211院校优先；

    2. 3年以上电商运营的工作经验，有过一个或多个电商平台的商城运营经验，运营过课程类产品最佳；

    3. 了解移动端商城的特点，熟悉有赞商城的运作方式为佳，有清晰的思路，统筹产品和销售策略；熟悉电子商务流程，具有良好的数据分析能力、数据库优化和管理，熟悉淘宝销售过程；

    4. 有较好的活动策划能力，能协调和利用各方面资源，针对不同的时期和不同的事件有效的进行活动的策划；

    5. 有较强的执行能力，能较好地把各种方案和计划落地操作；

    6. 有良好的沟通、团队协作能力；

    7. 熟悉自媒体，有垂直类电商工作经验优先。

  • 三. 产品研发部副总监

    • 工作职责：

    全面负责课程研发的规划制定和具体实施、课程开发师的培训和考核。

    1. 搭建课程体系、规划年度产品监管课程的制作、保证课程质量；

    2. 培训、考核和评估课程开发师的工作；

    3. 同线下产品部协调配合，制作相应的课程内容；

    4. 协同产品销售。

    职位要求：

    1. 211／985以上学校毕业或有海外留学经验，英语专八或有优异的出国考试成绩（托福110以上或雅思7.5以上）；

    2. 4年以上英语教育相关行业经历，有幼儿教育背景或在线教育背景优先；具有一线的而教学经验以及课程研发经验；

    3. 熟悉在线教育的发展状况，对于市场趋势，用户需求的变化具有较强的敏感性；在课程研发方面，负责过课程的标准化制作，熟悉从个性化到标准化过程当中的工作流程和注意事项，能够用合理的方案推进标准化进程；

    4. 具有带领团队的经验，具有团队建设、人员培养、成员激励等相关能力，带过10人以上的团队，并且在团队管理、培训方面有丰富的经验；

    5. 善于提取和总结规律，制定可实施的标准，在语言难度和课程丰富度方面能够推行标准化，并且制定可操作性的方案逐步实施；

    6. 具备系统思考能力、学习能力、沟通协调能力，擅于对用户反馈进行分析、总结，快速响应；

    7. 执行力和抗压力强，工作积极主动，有创业心态，及时响应。

  • 四. 产品研发部课程开发师

    • 工作职责：

    有一定的产品经理属性，负责1-2门课程的规划和具体实施。包括：

    1. 配合部门的产品规划，编写课表和课程大纲；

    2. 和外教协作形成每一节课的文本内容并安排负责录音的外教和中教完成录音工作；

    3. 负责和设计、插画沟通，完成课程当中相关图片的绘画和制作；

    4. 负责和视频组沟通，完成课程视频的制作；

    5. 负责和前端工程师沟通，完成随堂测试、阶段性测试的题目；

    6. 编写课程相关的印刷品（讲义、练习册、卡片）；

    7. 根据学员的反馈进行对课程调整。

    职位要求：

    1. 211/985以上学校毕业或有海外留学经验，英语专八或有优异的出国考试成绩(托福105以上或雅思7以上)；

    2. 2年以上英语教育相关行业经历，有幼儿教育背景或在线教育背景优先；具有课程研发经验；具备英语口语教学经验优先；

    3. 熟悉在线教育的发展状况，对于市场趋势，用户需求的变化具有较强的敏感性；能够快速学习新鲜事物；

    4. 必须参与过超过3个部门以上的项目协作经验，严格理解和掌握时间节点，并且可以充分进行团队沟通，有较强的协作能力；

    5. 擅于运用mind manager规划整体课程；既能够对课程知识点进行总结，又能够对重点知识展开详细讲解；对英语的发音特点、难点也有一定的了解；

    6. 具备系统思考能力、学习能力、沟通协调能力；对自己的产品严格要求；

    7. 执行力和抗压力强，工作积极主动，有创业心态，及时响应。

  • 五. 技术经理

    • 工作职责：

    1. 负责公司后端核心服务的设计和开发；

    2. 负责系统架构设计及性能优化；

    3. 负责前沿技术调研、团队管理和团队建设。

    职位要求：

    1. 计算机相关专业，本科及以上，211、985院校优先；

    2. 7年以上研发经验，3年以上技术管理经验，专业知识扎实，有丰富的分布式系统架构设计经验；

    3. 精通Java，熟悉Guice，Guava，Spring，Jetty，Redis，MySQL，Kafka等主流开源技术和框架；

    4. 有知名互联网公司工作经验优先，有独立负责的成功项目经验优先；

    5. 有很好的分析问题和解决问题的能力，责任心强，善于学习，善于沟通和表达，有良好的团队合作意识；

    6. 执行力和抗压力强，工作积极主动，有创业心态，及时响应并处理产品问题。

  • 六. 社群运营

    • 职位要求：

    1. 善于和用户打交道，处理用户问题并及时解决，收集用户反馈，做好社群内部的活动优化；

    2. 持续挖掘社群用户需求，探索更好满足用户需求的运营机制；建立并持续完善社群的工作流程以及制度规范；

    3. 定期进行社群用户数据统计分析，关注群员留存，转化；整合内外部资源，推广持续不断的运营手段，提升关注用户的活跃度和粘度。挖掘、维护核心用户，维护社群各方关系。

    任职资格：

    1. 本科及以上学历，具备1年以上新媒体运营或社群运营工作及管理经验；

    2. 有线上、线下活动策划组织经验，具有从零到一用户运营经验者优先；

    3. 具备一定的管理经验，善于调动资源、进行资源整合，善于发展及培养团队成员关系的能力；

    4. 具备良好的责任心、创新能力，沟通协调能力、执行力及团队合作精神，性格活泼，能调动社群气氛。

  • 七. 市场文案策划

    • 工作职责：

    1. 对公司教育类产品的上架文案撰写，挖掘产品卖点，吸引用户眼球；

    2. 协同设计师策划平台商品广告语和配图文字的撰写，给予商品描述内容的优化指示；

    3. 参与公司各种营销推广策划，并撰写策划方案和推广方案；

    4. 负责公司品牌和产品的宣传广告与新闻稿、软文等撰写。

    职位要求：

    1. 市场营销、中文、广告、新闻等相关专业本科以上学历；

    2. 熟悉在线教育的发展状况，对于市场趋势、用户需求的变化具有较强的敏感性；

    3. 有在线教育类产品及活动策划经验优先，有少儿教育、母婴行业背景优先；

    4. 有电子商务文案策划的经验，能用文字激励消费者购买欲的文案高手；

    5. 出色的沟通和语言表达能力，具有良好的团队合作能力；

    6. 擅长用推文吸引眼球，有过成功的微博微信拉新案例者优先。
